# 🎓 ANITS Campus Placement Portal

A responsive and user-friendly **Campus Placement Portal** developed using **HTML5 and CSS3**. This project simulates a college placement portal where students can register, log in, explore recruiting companies, and contact the placement department.

## 🚀 Features

- 🏠 Home page with a background video
- 📖 About Placement Department
- 📝 Student Registration Form
- 🔐 Student Login Page
- 🏢 Recruiting Companies Page
- 📞 Contact Page with Google Maps integration
- ✅ Registration Success Page
- 📱 Responsive design
- 🎨 Simple and clean user interface

---

## 🛠️ Technologies Used

- HTML5
- CSS3
